API绝对简单, 跨平台.

调用地址: http://zwc.name/IAmHere/API/submit.ashx

参数:
room, 房间的名称. 必须的.
name, 自己的名字
x,自己位置的x坐标(经度)
y, 自己位置的y坐标(纬度)
示例:
http://zwc.name/IAmHere/API/submit.ashx?room=root&name=曾文超&x=31.004999&y=121.4086

返回:
用户列表. 每行一个用户.每行3个数据, 用户名, x坐标, y坐标. 用逗号隔开.
示例:
zwc,1,2
zwc2,2,3
曾文超,31.004999,121.4086
